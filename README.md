SALES PREDICTION USING PCA AND LINEAR REGRESSION

OVERVIEW

This project builds a machine learning pipeline to predict sales based on advertising budgets for TV, Radio, and Newspaper. It applies Principal Component Analysis (PCA) to reduce feature dimensionality and improve model efficiency, followed by Linear Regression for prediction.

OBJECTIVES
Analyze relationships between advertising channels and sales
Apply feature scaling and PCA for dimensionality reduction
Train a regression model on transformed feature space
Evaluate model performance using standard metrics
DATASET

The dataset contains advertising budgets for three media channels (TV, Radio, Newspaper) and corresponding sales figures.

METHODOLOGY
Data loading and preprocessing
Feature scaling using standardization
PCA to reduce features to principal components
Linear regression model training on PCA-transformed data
Model evaluation using R² score, Mean Squared Error, and Mean Absolute Error
TOOLS AND TECHNOLOGIES
Python
NumPy
Pandas
Scikit-learn
Matplotlib
RESULTS

The model demonstrates that dimensionality reduction using PCA can still preserve predictive performance while simplifying the feature space.
